## Como usar?

1. Fazer o clone do repositório:
```bash
git clone https://github.com/MatheusWesley/todo-list.git
```

2. Após acessar a pastar baixada instalar as dependências com:
```bash
npm install
```
3. Executar projeto:
```bash
npm run dev
```
